# Classification de l'état célébrale d'un animal (Data Challenge)

L'objectif de ce projet est de classer l'état célébrale d'un animal. Pour cela, nous avons à notre disposition des séries temporelles représentant l’activité cérébrale d’un neurone. Dans ces fichiers, chaque individu correpond à un  ́echantillon de série et est constitué de 50 données prises dans le temps. L'étiquette de classification (”TARGET”) attribue la classe 1 pour l’état 1 et la classe 2 pour l’état 2. C’est donc un problème de classification binaire supervisée. Nous avons 16635 individus pour l’échantillon d’entrainement et 11969 individus dans l’échantillon test.
